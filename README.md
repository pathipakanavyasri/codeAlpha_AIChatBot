# codeAlpha_AIChatBot
AI ChatBot is a Java-based console application that simulates a simple chatbot. The chatbot interacts with users by responding to predefined questions and commands. It demonstrates the basic concepts of Artificial Intelligence through rule-based responses and natural language processing techniques.
# Features
Interactive conversation with users
Responds to common greetings and questions
Provides information about Java and Python
Handles unknown queries gracefully
Exit the chat using a command
Menu-free and user-friendly interface
# Technologies Used
Java
Object-Oriented Programming (OOP)
Scanner Class
String Handling
Conditional Statements
# Sample Output
Plain text
===== AI CHATBOT =====
Type 'bye' to exit.

You: hello
Bot: Hello! How can I help you?

You: what is java
Bot: Java is a popular object-oriented programming language.

You: how are you
Bot: I am fine. Thank you for asking!

You: bye
Bot: Goodbye! Have a nice day.
